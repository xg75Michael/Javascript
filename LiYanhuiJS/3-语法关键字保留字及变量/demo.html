Well Come to JavaScript World!
JS是一种弱类型语言，可以改变数据的同时改变类型；
区分大小写： text和Text是两种不同的表示符；
标识符组成开头必须是： a（字母） 或 _ 或 $；
标识符其他部分组成： a（字母） 或 _ 或 $ 或 数字；
不能把关键字， 保留字， true false null 作为标识符；
100 数字字面量；
'高向岩' 字符串字面量；
false 布尔值字面量 true;
/js/gi 正则表达式;
null 对象字面量;
{x: 1, y: 2} 对象字面量表达式；
[1,2,3] 数组字面量表达式；
关键字： 是已经使用的标识符；
保留字： 将来很可能使用的关键字；
